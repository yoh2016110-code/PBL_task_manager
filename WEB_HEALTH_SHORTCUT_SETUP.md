# Web版でショートカット同期を使う方法

Web版の「ショートカット同期」ボタンは、`GoalTaskHealth` という名前のショートカットを起動します。

## ショートカット名

ショートカット名を次にしてください。

```text
GoalTaskHealth
```

## 最後に追加する流れ

ヘルスケアJSONのテキストを作ったあと、次の順番にします。

1. JSONテキストを作る
2. `URLエンコード` を追加する
3. `ショートカットの入力` と、URLエンコードしたJSONをつなげる
4. `URLを開く` でそのURLを開く

## JSONの例

```json
{"date":"2026-07-09","steps":6460,"activeEnergy":179.382}
```

## 戻り先URLの形

Web版からショートカットを起動すると、ショートカットの入力として次のようなURLが渡されます。

```text
https://47k99ksf97-source.github.io/PBL_task_manager/?health=
```

ショートカットでは、この後ろにURLエンコードしたJSONをつけて開きます。

```text
ショートカットの入力 + URLエンコードしたJSON
```

これでWeb版に戻った時、自動でヘルスケア情報が読み込まれます。
